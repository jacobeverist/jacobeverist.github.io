## Past

### [BrainBlocks](https://github.com/jacobeverist/brainblocks)

A framework originally co-developed at The Aerospace Corporation for building scalable machine learning applications using principles derived from computational neuroscience. Models neuron activations with binary patterns (vectors of 1s and 0s) forming a "cortical language." Assemblages of computational blocks transmit these binary patterns to create workflows that exploit neuroscience principles.

### [Robot Mapping with Proprioception](https://github.com/jacobeverist/opaque)

Code repository containing the experiments and simulation framework for my PhD dissertation research: [*Exploring and Mapping Confined and Sensor-Challenged Environments with Proprioception of a Robotic Snake*](https://github.com/jacobeverist/opaque/blob/physical/doc/Everist_Dissertation_2015.pdf). Investigates how snake-like robotic probes can explore, map, and navigate unknown confined environments when all external sensors have failed, relying solely on proprioception (joint angles and body contact) and physical interaction with the environment.
